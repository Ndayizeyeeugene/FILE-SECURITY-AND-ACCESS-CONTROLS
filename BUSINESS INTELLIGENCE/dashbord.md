# Suggested Dashboards

## 1. Member Activity Dashboard
- Shows total actions per member
- Trend over time (daily, weekly)
- Top active members
- Cumulative actions using window functions

## 2. File Usage Dashboard
- Total files per member
- Public vs Private files distribution
- Recently added files
- Most accessed/edited files

## 3. Action Log Dashboard
- Actions by type (VIEW, EDIT, DELETE)
- Success vs Failed actions
- Last action per file
- Action timeline chart

## 4. Security & Compliance Dashboard
- Attempted deletions of ADMIN users
- Private file access attempts
- Rule violations logged

## 5. KPI Tracker Dashboard
- Displays KPIs such as total files uploaded, total actions, active members
- Alerts when thresholds are crossed
